1. JS 基础

介绍

​ JS 是运行在客户端（浏览器）的编程语言，实现人机交互效果，html 和 css 是标记语言。可以用于做网页特效、表单验证、数据交互、服务端编程（node.js）。

​ JS 由 ECMAScript 和 Web APIs 组成，其中 ECMAScript 规定了 js 基础语法，Web APIs 中的 DOM 操作文档（操作页面元素）、BOM 操作浏览器（操作页面弹窗）。
